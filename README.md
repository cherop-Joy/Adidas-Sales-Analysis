# Adidas-Sales-Analysis dashboard- Power BI

The project presents an interactive sales analytics dashboard developed using Power BI to analyse Adidas Sales performance across multiple business dimensions.They include time,geography,category of the product and retailers.The dashboard helps stakeholders understand KPIs and identify opportunities for improving and profitability
